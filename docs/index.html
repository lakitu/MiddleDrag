Get Started in Seconds

1 Install via Homebrew

brew tap nullpointerdepressivedisorder/tap && brew install --cask middledrag

2 Install via MacPorts

sudo port install middledrag

3 Grant Permissions

Like all gesture apps on macOS, MiddleDrag needs Accessibility permissions to see your trackpad touches.

System Settings > Privacy & Security > Accessibility > MiddleDrag

Or download the manual installer from GitHub Releases.
Requires macOS 15 Sequoia or later.
